OpenCV includes several functions for finding the size and shape of a detected object. To use them, we need to find the objects in the image. The best way to do this is to find all the regions of bright green. This will detect some things besides the target, but they are easier to remove later in processing. This is done with the function 

cv.inrange
, which uses an upper and lower bound for each channel (for example, in RGB, pixels could be rejected because of too much red, not enough red, etc). This requires the image to be in the necessary format already.
